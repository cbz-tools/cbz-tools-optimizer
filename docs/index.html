cbz-image-optimizer

Bulk-resize images inside ZIP/CBZ archives — significantly reduce file size with parallel processing.

Download Latest Release View on GitHub

Features

📦 Storage savings

Significantly reduce file size by resizing images to your target resolution.

⚡ Speed

Parallel processing across ZIPs and images via rayon.

🖥️ Cross-platform

Windows / Linux / macOS — single binary, no install required.

🖱️ GUI included

Windows drag-and-drop GUI — no CLI knowledge required.

CLI Usage

# Basic (default: ipad preset 2048x1536, JPEG quality 85)
cbz-image-optimizer input.cbz

# With options
cbz-image-optimizer --preset kindle --quality 80 *.cbz

# Custom size
cbz-image-optimizer --preset custom --max-width 1280 --max-height 720 input.zip
